# Curso em Vídeo -> Gustavo Guanabara
# Sobre
Aqui está visível toda a minha trajetória dentro do curso de Python do Gustavo Guanabara. Desde o primeiro "Hello, World!" até os desafios feitos com Programação Orientada a Objetos.

# MUNDOS
Os módulos do curso são denominados "Mundos", como um recurso de gamificação para essa jornada.

## Mundo 1
Fundamentos da linguagem, como: 
- Operadores aritméticos (+, -, *, /, **, ...)
- Funções básicas (print, input, int, float, str, bool, ...)
- Tratamento de strings (fatiamento, join, split, strip, ...)
- Utilização de bibliotecas básicas (math, time, random, ...)

## Mundo 2
Conceitos mais importantes e pertinentes de lógica de programação, como:
- Estrutura de decisão IF / ELSE 
- Tratamento de erros
- Variáveis compostas () Tuplas, [] Listas e {} Dicionários
- Introdução ao for e loops, como while True

## Mundo 3
Aprofundamento dos conceitos e introdução aos mais avançados, como:
- Estrutura de repetição for e while
- Manipulação avançada de variáveis compostas (como geração de matrizes e dados persistentes)
- Funções, modulação e docstrings (def, :params:, ...)
- Uso de bibliotecas como rich, requests e File I/O

## Mundo 4
O primeiro módulo que aborda Programação Orientada a Objetos de forma direta.
- O que é POO, estrutura e conceitos
- Herança
- Abstração
- Encapsulamento
- Polimorfismo 

## Exercícios
Os arquivos de exercícios são os exemplos apresentados pelo Gustavo Guanabara durante os vídeos, incentivando os alunos a praticarem antes de partir para os desafios.
